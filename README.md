# amazon-scrapper
API to get details of products from amazon.in

## Functions
- Get product details 

  Route: **`/products/:productId?api_key=`**
- Get Product Reviews

  Route: **`/products/:productId/reviews?api_key=`**
- Get Product offers

  Route: **`/products/:productId/offers?api_key=`**
- Get Search results
  
  Route: **`/search/:searchQuery?api_key=`**
